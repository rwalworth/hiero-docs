### HintsKeyPublicationTransactionBody
A transaction body to publish a node&#39;s hinTS key for a certain
party id and number of parties. A hinTS key is an extended
public key; that is, a BLS public key combined with &#34;hints&#34;
derived from the matching private key that a signature
aggregator can use to prove well-formedness of an aggregate
public key by an efficiently verifiable SNARK.


| Field | Type | Label | Description |
| ----- | ---- | ----- | ----------- |
| party_id | [uint32](#uint32) |  | The party id for which the hinTS key is being published; must be in the range [0, num_parties). &lt;p&gt; This value MUST be set to a non-negative integer.&lt;br/&gt; |
| num_parties | [uint32](#uint32) |  | The number of parties in the hinTS scheme. |
| hints_key | [bytes](#bytes) |  | The party&#39;s hinTS key. |
